Веб-приложение "Обучайщий сайт по КОМПАС - 3D"

### *Структура проекта*

Репозиторий содержит две основные части:

1. Фронтенд (`/compas-client/front`)
*   Технологии: Vue 3, Vite, CSS3.
*   Статус: Верстка главной страницы почти готова. В процессе разработка адаптивности, реализована интерактивная карусель (как предполагалось в макете)
*   Демо: [Посмотреть работающий сайт](https://tolstovakatya.github.io/compas-application/) *(Ссылка заработает после настройки GitHub Pages, будет готово после окончания верстки главной страниц)*

2. Бэкенд (`/compas-server`)
*   Технологии: Python, Django, Django REST Framework.
*   Структура:
    *   `/compas` — основное приложение Django (модели, представления, URL).
    *   `venv` — виртуальное окружение (не загружается в репозиторий).
    *   `requirements.txt` — список зависимостей для установки.
*   Статус:  Архитектура создана, модели данных готовы.



### *Локальный запуск*

Требования
*   Node.js (для фронтенда)
*   Python 3.8+ (для бэкенда)

*Запуск Фронтенда*

Откройте терминал в папке фронтенда

```bash
cd compas-client/front
npm install
npm run dev
```

После запуска сервер будет доступен по адресу: http://localhost:5173

*Запуск Бэкенда*

Откройте терминал в папке бэкенда

```bash
cd compas-server

# 1. Создать и активировать виртуальное окружение
python -m venv venv

# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Применить миграции и запустить сервер
python manage.py migrate
python manage.py runserver
```

После запуска сервер будет доступен по адресу: http://127.0.0.1:8000


### *Примечания*
*   Файлы .env, базы данных db.sqlite3 и папки node_modules/venv исключены из репозитория в целях безопасности и оптимизации.
*   Проект находится в активной разработке.
